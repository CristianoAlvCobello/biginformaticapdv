# 💳 Big Informática PDV - Landing Page

Landing page desenvolvida para apresentar o **sistema PDV da Big Informática**, destacando suas funcionalidades, equipamentos compatíveis e benefícios para a gestão de vendas de empresas e comércios.

O projeto foi desenvolvido com foco em **experiência do usuário, clareza na apresentação das soluções e conversão de clientes**, utilizando um layout moderno e responsivo.

---

## 🌐 Acesse o projeto

🔗 https://biginformaticapdv.com.br

---

## 📌 Sobre o projeto

Este site foi criado como uma **landing page institucional** para divulgar o sistema **PDV da Big Informática**, uma solução completa para gestão de vendas.

A página apresenta:

* Informações sobre o sistema
* Modelos de máquinas compatíveis
* Benefícios da solução
* Integrações e funcionalidades
* Contato direto com a empresa

O objetivo principal é **apresentar a solução de forma clara e atrativa para possíveis clientes**.

---

## 🚀 Tecnologias utilizadas

Este projeto foi desenvolvido utilizando:

* **HTML5**
* **CSS3**
* **JavaScript**
* **Tailwind CSS**
* **AOS (Animate On Scroll)** para animações
